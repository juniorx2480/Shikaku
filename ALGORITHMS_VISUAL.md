# Referencia Visual de Algoritmos

## Metodo usado
Backtracking con validacion temprana.

## Flujo
- Cargar grilla.
- Encontrar celdas con numero.
- Generar candidatos.
- Probar y retroceder cuando hay conflicto.

## Resultado esperado
- Solucion valida o respuesta sin solucion.

Este archivo se dejo en formato simple para evitar simbolos especiales.
